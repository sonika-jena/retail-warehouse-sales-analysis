# Retail & Warehouse Sales Analysis

## Project Overview
This project analyzes retail and warehouse sales data to uncover trends, channel dominance, and product performance using Python and Power BI.

## Business Use Case
The analysis helps sales managers and inventory planners understand which products and channels drive revenue, enabling better strategic decisions.

## Dataset
- 30,000 records
- 14 features
- Sales data across retail and warehouse channels

## Data Processing
- Cleaned raw transactional data
- Engineered features such as total sales and dominant sales channel
- Prepared analysis-ready dataset

## Exploratory Data Analysis
- Retail vs warehouse sales comparison
- Product category performance
- Supplier-level analysis
- Time-based sales trends

## Power BI Dashboard
An interactive dashboard was built to visualize KPIs, channel performance, and trends.

Dashboard previews are available in the `images/` folder.

## Key Insights
- Warehouse sales dominate total revenue.
- Wine is the highest-selling product category.
- Most items sell primarily through one channel.
- Sales trends show seasonal behavior.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
